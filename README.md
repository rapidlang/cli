# RAPID CLI [WIP]

Creates a nice development environment

- compiling
- reloading
- deploying
- adding dependencies

### Install

`npm i rapid-cli -g`

### Language Documentation

Full documentation is available on https://rapid-lang.org page.

### Development

Use start script to start the server, watch changes and auto re-compile sources.

`yarn start`
